开始

测试

  • 三种方法都可以通过实例来调用，但是静态方法和类方法无法访问实例属性，所以更改了tst.data仅对普通方法起了作用

区别

  • 普通方法不能通过类名调用，但是静态方法和类方法是可以的

总结

  • 普通方法,可以通过self访问实例属性
  • 类方法,可以通过cls访问类属性
  • 静态方法,不可以访问,通过传值的方式
