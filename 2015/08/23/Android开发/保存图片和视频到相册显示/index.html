保存照片和视频到相册显示

发表于 2015-08-23   |   分类于 Android   |     |   本文访问数 次

1. 前言

照片和视频保存到本地的方法大致都是通过流的方式写入文件里面就可以达到保存到文件夹的目的,但是你保存到文件夹的资源却不一定能够在相册显示出来，只能翻看文件管理。网上介绍的大多是发送广播扫描的方式，但是测试之后感觉不太完善，所以整理了一下保存照片后通知到相册显示的代码，亲测可用。

2. 保存到系统资源相册

Environment.getExternalStorageDirectory().getAbsolutePath() + "/DCIM/Camera";保存到系统相册或者一些手机制定好的存储影音资源的文件夹，可以直接发广播扫描就可以达到添加到相册的目的。

3. 保存到非系统资源相册

可能我们的应用需要将图片和视频保存到自己app的文件夹，使用广播扫描就无法扫描到了，可以采用MediaStore 直接插入的方式写到ContentProvider中。

4. 注意

保存到系统相册等可以扫描到的相册，只需要发广播，不能再insert了，否则会出现两张一样的照片。

在应用过程中发现，vivo手机和魅族手机部分机型只支持在文件管理中查看视频，使用本文描述的方法添加后没有效果，用微信保存视频试了一下同样不能在相册中将视频显示出来，应该是手机的原因。
